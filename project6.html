About Bookish

Independent Project

Solely designed and developed a dynamic book discovery and comparison website with a focus on clean UI, responsive layout, and user-driven features.

User-Centric Design

Built an intuitive interface to help users search, discover, and organize books with advanced filtering by title, author, and ISBN.

Personal Library & Profile Management

Enabled users to save favorite books, manage a personal library, and edit profile details including avatar selection and preferences.

Firebase Integration

Utilized Firebase Authentication and Firestore to manage user accounts, persistent storage, and real-time updates across devices.

Third-Party Integrations

Implemented EmailJS for contact form handling, and Dicebear for profile avatar generation to enhance interactivity and personalization.
